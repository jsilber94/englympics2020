#Commands

run app: ```python app.py``` \
Activate env (optional): ```./env/Scripts/activate.bat``` \
Install dependencies: ```pip install -r requirements.txt``` \
Make requirement.txt from dependencies inside env: ```pip freeze > requirements.txt```
